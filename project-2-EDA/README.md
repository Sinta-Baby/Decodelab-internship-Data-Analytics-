# Project 2 - Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on the e-commerce dataset to identify patterns, trends, outliers, and business insights.

## Tasks Performed
- Dataset Inspection
- Missing Value Analysis
- Duplicate Record Check
- Statistical Analysis
- Product Analysis
- Payment Method Analysis
- Referral Source Analysis
- Monthly Sales Trend Analysis
- Outlier Detection
- Correlation Analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- VS Code

## Key Findings
- Online payment method was most preferred.
- Instagram generated the highest referral traffic.
- Printer was the best-selling product.
- Sales peaked in June 2024.
- Strong positive correlation exists between UnitPrice and TotalPrice.

## Output
Visualizations and business insights generated through Exploratory Data Analysis.
