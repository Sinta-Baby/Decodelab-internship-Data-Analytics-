## Conclusion

This project successfully demonstrated SQL-based data analysis on an e-commerce dataset. A SQLite database was created, the dataset was imported, and SQL queries were executed to retrieve and analyze business information. Operations such as **SELECT**, **WHERE**, **ORDER BY**, **GROUP BY**, **COUNT()**, **SUM()**, **AVG()**, and **LIMIT** were used to extract valuable insights, including total orders, revenue, average order value, product performance, payment methods, and order status distribution. The project strengthened practical SQL querying skills and showed how SQL can be used to support business decision-making through data analysis.
