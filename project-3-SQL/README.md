# Project 3: SQL Data Analysis

## Overview

This project demonstrates SQL-based data analysis using an e-commerce sales dataset. The dataset was imported into a SQLite database, and SQL queries were used to extract meaningful business insights.

## Objectives

* Create a SQLite database
* Import CSV data into a SQL table
* Perform data retrieval and analysis using SQL
* Generate business insights from transactional data

## Dataset Information

* Dataset Type: E-Commerce Sales Data
* Total Records: 1200
* Total Columns: 14

## Tools & Technologies

* SQLite
* SQL
* VS Code
* SQLite Viewer

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* SUM()
* AVG()
* LIMIT

## SQL Analysis Performed

* Retrieved all records
* Filtered records using WHERE
* Sorted records using ORDER BY
* Grouped records using GROUP BY
* Counted total orders
* Calculated total revenue
* Calculated average order value
* Identified top-selling products
* Analyzed payment methods
* Analyzed order statuses
* Identified customers using coupons
* Retrieved highest-value orders
